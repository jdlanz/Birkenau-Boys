# Birkenau-Boys
This repository contains data, analysis scripts, and supporting materials for the study of a group of Holocaust survivors known as the Birkenau Boys.

# About the Project
Data for this project was collected as part of a doctoral dissertation completed at Indiana University Bloomington, USA. It is currently being analyzed as part of an in progress book project which seeks to write a history of the 'Birkenau Boys,' a group of 89 child Holocaust survivors. All data analysis for the project is conduced in RStudio.

# Data Sources 
All data in this repository was sources from the following archives: United States Holocaust Memorial Museum, State Museum at Auschwitz-Birkenau, Jewish Museum Berlin, Yad Vashem, The Wiener Library, and Beit Terezin. Additional biographical data was sourced from John Freund, Ed., _After those Fifty Years: Memories of the Birkenau Boys._ Toronto: Self-published, 2008. Specific archival collections and citations will soon be uploaded to this README.
